# Slot-Machine Vite

A modularized slot machine build by Vue3 + Vite with optional and SFC api.

## development

pnpm install
pnpm run dev

## build

pnpm build

## Recommended IDE Setup

- [VS Code](https://code.visualstudio.com/) + [Volar](https://marketplace.visualstudio.com/items?itemName=Vue.volar)
